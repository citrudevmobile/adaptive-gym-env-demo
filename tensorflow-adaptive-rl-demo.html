What is Adaptive Reinforcement Learning?

Normal RL: The rules never change. The agent learns ONE strategy.

Adaptive RL: The rules CHANGE over time. The agent must KEEP LEARNING.

In this game, the hill moves every N steps, and N changes randomly between episodes!

What is a Deep Q-Network (DQN)?

Instead of a big lookup table (Q-table), DQN uses a NEURAL NETWORK to predict Q-values.

Benefits: Handles continuous states, generalizes to new situations, scales to complex problems.

The Bellman Equation

Q(s,a) = Q(s,a) + α * [ R + γ * max(Q(s',a')) - Q(s,a) ]

This is how the agent learns from rewards and updates its neural network.

Key DQN Features in This Code

  • Experience Replay: Stores past experiences, learns from random batches
  • Target Network: Separate network for stable training targets
  • Epsilon-Greedy: Balances exploration vs exploitation
  • Adaptive Epsilon: Explores more when performing poorly
  • Adaptive Environment: Hill move speed changes over time

Reading the Metrics

  • Episode Reward: Total points in current game (higher = better)
  • Epsilon: 0.3 = 30% random actions, 70% learned actions (decreases over time)
  • Network Loss: How wrong the network is (lower = better)
  • Avg Adaptation Time: How many steps to find hill after it moves (lower = better)
  • Move Interval: How many steps until hill moves again (changes adaptively!)

What to Watch For

  • Episode Reward should INCREASE over time (agent is learning)
  • Epsilon should DECREASE over time (agent explores less, exploits more)
  • Network Loss should DECREASE over time (predictions get better)
  • Adaptation Time should DECREASE (faster reaction to hill moves)
  • Move Interval changes randomly - watch the agent adapt!
